## 二、核心功能模块

### 2.1 登录模块

**功能描述**：用户通过账号和密码登录系统

**功能需求**：

1. 支持账号登录
2. 密码显示/隐藏切换
3. 记住账号功能
4. 根据账号自动识别用户角色
5. 登录后跳转至对应权限的主页面

**UI要素**：
- 背景图：二手车相关主题图
- 系统Logo和名称
- 登录表单（账号、密码、记住账号、登录按钮）

---

### 2.2 抖音AI小高客服（抖音会话）

**功能描述**：接入抖音企业号，实现AI自动托管回复客户咨询

**前置条件**：需先完成抖音企业号授权接入

**功能需求**：

#### 2.2.2 抖音企业号列表

##### 展示字段：

- 企业号头像
- 企业号昵称
- 未读消息数

#### 2.2.3 客户会话列表
**展示字段**：

- 客户头像
- 客户姓名
- 最后消息内容
- 消息时间
- 在线状态（在线/离线）
- 消息标签：
  - 需人工
  - 高意向
  - 已留资
  - 待回访
- 未读消息数

**交互功能**：

- 点击切换会话
- 搜索客户
- 按标签筛选

#### 2.2.4 聊天面板
**消息类型**：

- 用户消息：客户发送
- AI消息：AI自动回复
- 人工消息：客服手动回复

**托管模式**：

1. **AI托管模式**：
   - 自动根据车型和留资意向回复
   - 显示"正在根据车型和留资意向自动回复"提示
   - 蓝色状态指示灯

2. **人工接管模式**：
   - 客服手动输入回复
   - 支持 Enter 键发送，Shift+Enter 换行
   - 黄色状态指示灯
   - 可恢复AI托管

**工具栏**：
- 表情
- 图片
- 视频
- 文件

#### 2.2.5 客户信息面板
**展示信息**：
- 客户姓名、头像、在线状态
- 来源渠道
- 意向车型
- 年份
- 预算范围
- 所在城市
- 当前标签
- 溯源信息

**线索评分**：
- 百分比展示（0-100）
- 进度条可视化

---

### 2.3 AI小高线索（线索管理）

**功能描述**：聚合AI识别、留资和销售跟进状态，统一管理销售线索

**页面路径**：`/leads`

#### 2.3.1 统计卡片
| 指标 | 说明 | 趋势提示 |
|-----|------|---------|
| 累计线索 | 历史总线索量 | 较昨日增长率 |
| 已分配 | 已分配给销售的线索 | 销售响应率 |
| 已留资 | 客户已留下联系方式 | 转化率 |
| 高意向 | 高意向客户数量 | 需优先跟进 |

#### 2.3.2 线索列表
**筛选条件**：
- 关键词搜索（联系人、车型、电话）
- 来源筛选
- 状态筛选
- 重置筛选

**列表字段**：
- 联系人（头像+姓名+城市）
- 线索信息（来源·车型·预算·创建时间）
- 联系电话
- 溯源信息
- 状态
- 分配销售
- 操作（重新分配/对话跟进按钮）

**状态流转**：

- 新线索 → 跟进中 → 已留资 → 已成交/已失效

#### 2.3.3 线索详情面板
**客户信息**：
- 头像、姓名、状态标签
- 手机号、来源、意向车型、预算、城市、当前销售
- 溯源信息

**线索评分**：

- 百分比显示
- 进度条可视化

**操作按钮**：

- 重新分配
- 对话跟进

**销售跟进记录**：

- 时间线展示
- 操作内容
- 备注信息
- 操作时间

**重新分配弹窗**：
- 选择接待销售
- 分配备注
- 确认分配

### 2.5 抖音企业号管理（绑定账号）

#### 2.5.1 抖音企业号授权

1. 点击"添加抖音企业号"按钮
2. 获取授权链接
3. 展示二维码供用户扫码授权
4. 授权成功后同步账号信息

**授权信息**：

- 企业号名称：如"老高二手车企业号"
- 企业号头像：抖音头像
- OpenID：抖音平台唯一标识
- Key：系统内部分配的密钥

#### 2.5.2 抖音企业号列表

- 企业号名称：如"老高二手车企业号"
- 企业号头像：抖音头像
- 企业号标签
- 抖音号：123456
- 会话授权状态：已授权/未授权
- 绑定智能体：专业金牌顾问
- 操作：选择智能体/取消授权/删除

### 2.6 AI小高助手（微信代理）

**功能描述**：下载并启动微信客户端，配置自动执行规则

**页面路径**：`/ai-agent`

#### 2.6.1 微信客户端管理
**客户端状态**：
- 未启动
- 未登录
- 已登录

**建议安装路径**：`C:\Program Files\Tencent\WeChat`

**功能按钮**：
- 立即下载（微信客户端）
- 刷新（刷新客户端状态）

#### 2.6.2 配置列表
**配置信息**：
- 微信昵称
- 绑定规则（可多个）
- 负责人
- 联系电话
- 配置状态（启用/禁用）

**交互功能**：
- 添加配置
- 搜索配置（按昵称、规则）
- 切换启用/禁用
- 启动微信测试

**添加配置弹窗**：
- 绑定微信账号
- 规则配置（多选）
- 负责人
- 联系电话

**规则类型**：
- 素材自动发送

- 线索自动录入

#### 2.6.3 统计数据
- 已分配线索
- 进行中线索
- 已成交线索

#### 2.6.4 历史记录
**记录内容**：

- 执行结果（成功/失败）
- 执行时间

### 2.7 小高算力（算力中心）

**功能描述**：查看Token余额、消耗统计和充值

**页面路径**：`/compute`

#### 2.7.1 余额展示
- 算力余额（Token数）
- 立即充值按钮

#### 2.7.2 统计卡片
| 指标 | 说明 |
|-----|------|
| 今日消耗 | 今日Token消耗量 |
| 昨日消耗 | 昨日Token消耗量 |
| 累计消耗 | 历史总消耗量 |

#### 2.7.3 Token明细
**表格字段**：
- 类型（充值/消耗）
- Token变动（+增加/-消耗）
- 备注
- 时间

**收支颜色**：
- 收入：绿色 text-emerald-600
- 支出：深灰色 text-[#475467]

#### 2.7.4 算力充值
**充值弹窗**：

**充值方式**：
1. 套餐充值：预设套餐选择
2. 自定义金额：输入充值金额

**套餐示例**：
| 套餐名称 | 价格 | Token数量 |
|---------|------|----------|
| 基础版 | 99元 | 100,000 Token |
| 标准版 | 299元 | 350,000 Token |
| 专业版 | 699元 | 900,000 Token |

**支付方式**：
- 微信支付
- 支付宝

**充值流程**：
1. 选择套餐或输入金额
2. 选择支付方式
3. 生成付款码
4. 显示订单号
5. 完成支付

## 三、超级管理员功能

### 3.1 商户管理

**功能描述**：管理商户账号、功能授权、违禁词、回访提示词、AI回复记录、算力余额

**页面路径**：超级管理员专属

#### 3.1.1 商户列表
**筛选条件**：
- 商户名称搜索
- 账号搜索
- 智能体筛选
- 状态筛选
- 重置筛选

**列表字段**：
- 登录账号
- 商户名称
- 门店名称
- 类型（多选）
- 有效期（延期按钮）
- 功能授权（多个）
- 算力余额（充值按钮）
- 状态（启用/禁用）
- 创建时间

#### 3.1.2 新增商户
**必填字段**：
- 商户名称
- 登录账号
- 类型（多选）
- 功能授权（多选）

**可选字段**（创建后配置）：
- 有效期
- 算力额度
- 启用状态

**分类选项**：

- 精品代步车
- 金融方案助手
- 检测报告讲解

**功能授权选项**：
- 抖音客服
- 自动分配线索
- 每日报表推送

#### 3.1.3 编辑商户
**可编辑字段**：

- 商户名称
- 分类
- 功能授权

**不可编辑**：
- 登录账号（创建后固定）
- 创建时间

#### 3.1.4 商户操作
**延期**：
- 输入延期天数
- 累加到现有有效期

**充值**：
- 输入Token数量
- 累加到现有余额

**发放套餐**：
- 选择套餐
- 显示发放Token数
- 显示发放后余额

**启用/禁用**：
- 切换商户状态

**重置密码**：
- 重置商户登录密码

1. **输入框**
